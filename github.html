¿Que es GitHub?

GitHub es una plataforma web que permite almacenar, gestionar y compartir código fuente de proyectos de software.

¿Para que sirve GitHub?

Funcion Descripción
Almacenar código Guardar proyectos de forma segura en la nube
Control de versiones Ver historial de cambios y volver a versiones anteriores
Trabajo colaborativo Varios programadores trabajan en el mismo proyecto sin conflictos
Seguimiento de cambios Saber quién modificó qué y cuándo
Revisión de código Revisar y aprobar cambios antes de integrarlos
Gestión de proyectos Organizar tareas, bugs y mejoras
Documentación Crear wikis y páginas explicativas
Automatización Ejecutar pruebas y despliegues automáticos
Portafolio profesional Mostrar tus proyectos a empleadores

¿En que casos se usa GitHub?

  • Aprendiendo a programar guardando ejercicios y proyectos de practica
  • Creando un proyecto propio en el que se desarrole una app o pagina web personal
  • Haciendo copia de seguridad el cual permitira que el codigo este seguro en caso de presentar fallas en tu computadora
  • Llevando tu código a cualquier lado
  • Trabajo entre compañeros sin la mezcla de proyectos
  • Delegacion de funciones el cual uno programa, otro revisa y aprueba
  • Documentación compartida en donde todos logran ver las mismas versiones actualizadas en tiempo real
  • Publicar tu proyecto gratis de tal manera que cualquiera puede usar o mejorar tu codigo
  • Contribuir a proyectos existentes
  • Aprender de otros
  • Construir reputación
  • Gestión de código privado
  • Integración con otras herramientas
  • Automatización de despliegues
  • Control de accesos en donde decides quien puede ver o modificar cada parte
  • Profesores comparten ejemplos
  • Revisión de tareas
  • Proyectos grupales o colaborativos
  • Tutoriales y cursos
