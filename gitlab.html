¿Qué es GitLab

es una plataforma web de desarrollo colaborativo basada en Git que ofrece un enfoque todo-en-uno (all-in-one) el cual incluye herramientas integradas de CI/CD (Integración Continua/Despliegue Continuo) de forma nativa.

¿Para qué sirve GitLab?

Función Descripción
Control de versiones (Git) Gestión de código fuente con historial completo
CI/CD Integrado Automatización de pruebas, builds y despliegues
Gestión de proyectos Issues, milestones, tableros Kanban, epics
Registro de contenedores Almacena imágenes Docker
Seguridad DevSecOps Escaneo de vulnerabilidades en código y dependencias
Monitoreo y métricas Performance monitoring, logs, tracing
Wiki y documentación Documentación integrada por proyecto
GitLab Pages Hosting estático similar a GitHub Pages
Infrastructure as Code Gestión de infraestructura mediante código

¿En qué casos se usa GitLab?

  • Guardar ejercicios y proyectos de práctica en la nube
  • Ver el historial de cambios y cómo evoluciona tu código
  • Recuperar versiones anteriores si algo sale mal
  • Acceder a tu código desde cualquier computadora
  • Mostrar proyectos personales a posibles empleadores
  • Documentar qué hace cada proyecto con README
  • Tener un respaldo profesional de tu trabajo
  • Compartir tu código con la comunidad
  • Colaborar con compañeros de curso o bootcamp
  • Ver quién hizo qué cambio y cuándo
  • Evitar que se pierda el trabajo del equipo
  • Revisar el código de otros antes de aceptarlo
  • Automatizar tareas repetitivas sin configurar herramientas externas
  • Ver cómo se ejecutan pruebas automáticamente
  • Desplegar tu página web automáticamente al hacer cambios
  • Entender DevOps con una sola plataforma
  • Repositorios privados ilimitados gratis
  • Experimentar sin miedo a que otros vean tu código inicial
  • Practicar con proyectos personales antes de publicarlos
  • No depender de tu computadora para guardar el código
