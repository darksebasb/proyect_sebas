¿Qué es CSS?

Es un lenguaje de diseño usado para darle la apariencia visual a una página web y controlar los aspectos como: color, tipografía, márgenes y animaciones.

¿Para qué sirve CSS?

  • Separar la estructura del documento HTML.
  • Permite realizar cambios en la apariencia de una web sin afectar de manera dramática a su contenido.
  • El rendimiento de la página web es más ágil y eficaz.

¿En qué casos se usa CSS?

Se usa CSS en prácticamente cualquier proyecto web para transformar el código HTML sin estilo en una interfaz atractiva, organizada y funcional.

  • Diseño y Maquetación: Define la estructura de la página como: diseños de columnas y posicionamiento de elementos.
  • Estilo de texto y fuente: Personalización de colores, tamaños de letras y espacio de párrafos y encabezados.
  • Adaptabilidad: Crear sitios web que se adapten al tipo de dispositivo donde se esté haciendo la apertura de estas páginas o sitios web (móvil, tablet, computador, etc.).
  • Animaciones y Transiciones: Crear efectos visuales que contengan transiciones o animaciones que mejoren la interacción con los usuarios.
  • Uniformidad en el sitio: Aplicando los estilos de manera uniforme en múltiples páginas desde un mismo archivo CSS.
  • Estilos de Formulario y Multimedia: Mejora en la apariencia de botones y ajuste en el estilo de imágenes y videos.
