¿Que es una base de datos?

es un software informático que permite almacenar grandes cantidades de información, de una forma organizada y accesible para su futuro uso

¿Para que sire una base de datos?

sirve para almacenar, organizar, gestionar y recuperar grandes volúmenes de información de forma estructurada, rápida y segura

¿Que es data first y code first?

Data First: es un método de trabajo donde la base de datos ya existe o tú la creas primero manualmente (usando tablas tipo Excel).

Code first: Es un método de trabajo donde tú defines cómo quieres que sean tus datos escribiendo clases en tu lenguaje de programación

¿Que es un DTO y una entidad?

La Entidad es el objeto "real" que vive en tu base de datos. Tiene todos los detalles, incluso los que nadie ve.
El DTO (Data Transfer Object) Es una versión personalizada de la entidad transportar datos de forma segura y limpia hacia afuera de tu sistema.

¿Cuál es la diferencia entre DTO y ENTIDAD?

Característica Entidad (El Almacén) DTO (El Paquete)
Privacidad Tiene datos sensibles (contraseñas, IDs internos). Solo tiene datos públicos o necesarios.
Uso La usa el ORM para hablar con la base de datos. La usa la App para mostrarle cosas al usuario.
Forma Es idéntica a la tabla de la base de datos. Puede combinar datos de varias tablas en una sola "bolsa".
