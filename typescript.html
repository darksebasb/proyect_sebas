¿Que es tyypescript?

es un lenguaje de programación que funciona como un "mejorador" de JavaScript

¿Para que sirve typescrip?

Permite identificar los pequeños errores de forma rapida de tal manera que no sea tan complejo a la hora de programar y a su vez proporciona sugerencias de autocompletado ya que entiende lo que estamos construyendo en el codigo.

¿En qué casos se usa typescript?

Su uso destaca especialmente en situaciones donde la seguridad y el orden son prioritarios.

cuando varias personas estan trabajando en el mismo codigo exige que se siga el mismo orden evitando que cuando se presente algun error por parte de algun integrante no se logre identificar y dañe el codigo

Se utiliza masivamente para crear sitios web con mucha interactividad. Es la opción preferida (o incluso obligatoria) al trabajar con herramientas modernas como:

  • Angular: Está construido totalmente con TypeScript.
  • React y Vue: Aunque permiten JavaScript, la mayoría de las empresas usan TypeScript para que sus componentes sean más fáciles de entender y reutilizar.

Se usa mucho en el lado del servidor con Node.js o Deno. Frameworks como NestJS lo utilizan para crear sistemas bancarios, tiendas en línea o redes sociales que necesitan ser muy estables y procesar datos de forma segura.

A través de herramientas como React Native o Ionic, los desarrolladores usan TypeScript para crear aplicaciones que instalas en tu teléfono (Android o iOS)

Si alguien crea una herramienta para que otros programadores la usen, suele escribirla en TypeScript. Esto permite que el usuario final reciba ayuda automática del editor (autocompletado) y sepa exactamente qué datos debe enviar a cada función.
