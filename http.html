¿Qué es HTTP?

Es el "lenguaje" que usan tu navegador y los servidores para comunicarse.

Métodos HTTP Principales

Método Propósito Uso Típico
GET Leer/Obtener datos Consultar información
POST Crear nuevos datos Agregar un recurso
PUT Actualizar datos Reemplazar todo un recurso
DELETE Eliminar datos Eliminar un recurso específico
PATCH Actualizar parcialmente datos Modificar solo algunos campos de un recurso
HEAD Obtener solo los encabezados Verificar la existencia de un recurso sin descargarlo
OPTIONS Consultar métodos soportados Descubrir qué métodos HTTP admite un recurso
CONNECT Establecer un túnel de comunicación Usado principalmente para conexiones seguras (HTTPS)
TRACE Rastrear la ruta de una solicitud Depuración y diagnóstico de la ruta de una solicitud
