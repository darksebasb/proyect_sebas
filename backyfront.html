¿Cual es la diferencia entre back y front?

Imagina que vas a un restaurante. Esa es la mejor forma de entender la diferencia entre el Frontend y el Backend:

El Frontend (La zona del comedor)

Es todo lo que tú ves y con lo que interactúas.

  • En el restaurante: Son las mesas decoradas, la carta con fotos de la comida, las luces, las sillas cómodas y el camarero que te sonríe.
  • En una página web: Son los colores, los botones que cambias de color al pasar el ratón, las imágenes, el tipo de letra y dónde están puestos los menús.
  • ¿Qué se usa aquí? Principalmente lenguajes como HTML, CSS y JavaScript (o sus mejoras como TypeScript y frameworks como React).

El Backend (La cocina)

Es todo lo que pasa detrás de escena para que el servicio funcione, pero que tú nunca ves directamente.

  • En el restaurante: Es donde los cocineros preparan la comida, donde está la despensa con los ingredientes (la base de datos), donde se lavan los platos y donde el administrador lleva las cuentas de cuánto dinero entró.
  • En una página web: Es el servidor que guarda tu contraseña de forma segura, el que decide si tienes permiso para ver una foto privada o el que procesa el pago cuando compras algo.
  • ¿Qué se usa aquí? Lenguajes como Python, Java, PHP o Node.js (que usa JavaScript en el servidor).
